#

## Installation

```
$ npm install
```

```
$ tsc -p .
```

```
$ node app.js
```

After go on the url `http://localhost:3000` and enter your pseudo.
Chose a room and wait max player :)

## Add new cards

All cards are in french, you can add cards in the `src/datas/cards.json` file.
